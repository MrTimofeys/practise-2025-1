Аннотация проекта

AlgoTrade Framework - это набор программ для эффективной работы в сфере алгоритмической торговли, которые работают с российскими биржами. Наш фреймворк предоставляет:

  • Унифицированный API для подключения к различным биржам
  • Встроенные инструменты для бэктестинга стратегий
  • Библиотеку популярных технических индикаторов
  • Систему управления рисками
  • Визуализацию торговых операций

Проект разрабатывается командой студентов в рамках дисциплины "Проектная деятельность".

Ключевые особенности

Модульность

Гибкая архитектура позволяет добавлять новые компоненты без изменения ядра системы.

Бэктестинг

Тестирование стратегий на исторических данных перед использованием на реальных рынках.

Риск-менеджмент

Встроенные механизмы для контроля рисков и защиты капитала.
