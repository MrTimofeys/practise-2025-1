О проекте

AlgoTrade Framework - это инновационное решение для разработки и выполнения алгоритмических торговых стратегий, созданное в рамках дисциплины "Проектная деятельность".

Цели проекта

  • Создание набора программ для алгоритмической торговли
  • Упрощение процесса разработки и тестирования торговых стратегий
  • Предоставление инструментов для анализа эффективности стратегий
  • Возможность торговли на российский биржах в реальном времени на основе полученных стратегий

Технологический стек

Бэкенд

  • Python 3.9+
  • backtrader (создания стратегий)
  • Pandas (для обработки данных)
  • NumPy (для математических вычислений)

Фронтенд

  • React.js (для визуализации данных)
  • Chart.js (для построения графиков)
  • Bootstrap (для адаптивного дизайна)

Инфраструктура

  • Docker (для контейнеризации)
  • PostgreSQL (для хранения данных)

Архитектура системы

Фреймворк построен по модульной архитектуре, состоящей из следующих основных компонентов:

  1. Ядро системы - отвечает за управление стратегиями, обработку данных и выполнение торговых операций.
  2. Адаптеры бирж - обеспечивают подключение к различным торговым площадкам через их API.
  3. Модуль бэктестинга - позволяет тестировать стратегии на исторических данных.
  4. Визуализация - предоставляет инструменты для анализа результатов торговли.
  5. Управление рисками - контролирует уровень риска и соблюдение установленных лимитов.
